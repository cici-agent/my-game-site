# 小菜鸟网站 - 完整需求文档
> 最后更新：2026-05-17

---

## 一、项目概述

- **网站名称**：小菜鸟
- **域名**：cici2.fun
- **定位**：专为10岁左右小朋友设计的游戏发布和开发平台
- **核心特色**：用语音 + AI 就能做游戏，不需要懂代码
- **技术栈**：HTML/CSS/JavaScript + GitHub + Vercel
- **GitHub仓库**：cici-agent/my-game-site

---

## 二、页面结构

| 页面 | 文件名 | 说明 |

## 五、核心模块详细需求

### 5.1 用户系统
- 用户注册 / 登录
- 个人主页：头像、金币余额、发布的游戏、玩过的游戏
- 头像可用金币购买

### 5.2 金币体系
**来源：**
1. 线上充值（真实支付，微信/支付宝，优先级低，后期开发）
2. 完成系统任务（打卡等）
3. 发布游戏（按发布数量 + 被玩人数获得）

**用途：**
1. 购买用户头像
2. 玩别人的游戏（按时长计费，发布者可自定义价格，如每5分钟消耗X金币）
3. 赠送金币给其他用户

### 5.3 游戏模块
**游戏来源：**
- 注册用户自己发布（上传HTML5 zip包）
- 管理员在后台发布

**游戏分类：**
射击类、塔防类、学习类、跑酷类、休闲类、养成类、RPG类（可扩展）

**发布者后台：**
- 游戏上线 / 下线
- 查看玩的人数
- 查看用户点评

**用户互动：**
- 对游戏写点评、评分

### 5.4 在线游戏编辑器
- 语音输入 → AI理解 → 自动生成/修改游戏代码
- 实时预览游戏效果
- 提供数据库和服务器资源
- 发布游戏按钮
- 初期面向小范围内测（班级朋友，5-10人）
- AI使用DeepSeek API（成本低）

### 5.5 搜索与排序
**搜索：**
- 按游戏名称搜索
- 按游戏类型筛选

**排序逻辑（可迭代）：**
- 评分高的排前面
- 热门（玩的人数多）排前面
- 新游戏独立板块 + 排序权重加成（保证新游戏曝光）
- 长期无人玩的游戏自动隐藏

### 5.6 内容审核
- 用户举报违规游戏
- 管理员审核新游戏上线

### 5.7 通知系统
- 有人玩了你的游戏
- 有人给你点评
- 金币变动通知

### 5.8 管理员后台
- 管理所有用户
- 审核游戏上线/下线
- 管理金币
- 查看平台数据
